# Trigger Cases

## Positive Triggers

- "Generate a source/successor differential overlay for this Discovery review."
- "The overlay labels cover peaks; fix the plot."
- "Why does this smoothed trace not match the raw signal?"
- "Create a clean tutorial chromatogram figure for the guide."
- "Review whether this PNG overlay can support product-gate evidence."

## Negative Triggers

- "Decide whether to promote the product gate."
- "Write the Gallery HTML layout."
- "Choose which generated PNGs should be tracked."
- "Run a RAW validation command."

## Near Neighbors

- Use `xic-human-review-gallery` for Gallery/HTML review UX around the figure.
- Use `xic-product-gate-advancement` for readiness or authority decisions.
- Use `xic-validation-artifact-retention` for version-control boundaries.
- Use `xic-raw-validation` when the issue is launching or accepting RAW-backed
  evidence.
