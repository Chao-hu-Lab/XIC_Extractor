# XIC Extractor GUI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox syntax for tracking.

**Goal:** Build a PyQt6 single-page desktop GUI that replaces manual CSV editing and run_pipeline.bat with four vertical sections: Settings, Targets, Run, Results.

**Architecture:** A MainWindow contains a QScrollArea stacking four QWidget sections. A PipelineWorker(QThread) runs the existing PS1 + Python scripts as subprocesses and emits typed signals. All CSV I/O is centralised in config_io.py. The existing scripts are not modified.

**Tech Stack:** Python 3.10+, PyQt6 6.6+, pytest, pytest-qt

```python
import csv
from pathlib import Path

ROOT = Path(__file__).parent.parent
CONFIG_DIR = ROOT / "config"
_SETTINGS_FIELDS = ["key", "value", "description"]
_TARGETS_FIELDS = ["label","description","mz","ms_level","rt_min","rt_max",
                   "ppm_tol","neutral_loss_da","nl_ppm_warn","nl_ppm_max"]

def read_settings() -> dict[str, str]:
    with open(CONFIG_DIR / "settings.csv", newline="", encoding="utf-8-sig") as f:
        return {r["key"]: r["value"] for r in csv.DictReader(f)}

def write_settings(settings: dict[str, str]) -> None:
    path = CONFIG_DIR / "settings.csv"
    with open(path, newline="", encoding="utf-8-sig") as f:
        existing = {r["key"]: r for r in csv.DictReader(f)}
    with open(path, "w", newline="", encoding="utf-8-sig") as f:
        w = csv.DictWriter(f, fieldnames=_SETTINGS_FIELDS)
        w.writeheader()
        for k, v in settings.items():
            w.writerow({"key":k,"value":v,
                "description":existing.get(k,{}).get("description","")})

def read_targets() -> list[dict[str, str]]:
    with open(CONFIG_DIR / "targets.csv", newline="", encoding="utf-8-sig") as f:
        return list(csv.DictReader(f))

def write_targets(targets: list[dict[str, str]]) -> None:
    with open(CONFIG_DIR / "targets.csv", "w", newline="", encoding="utf-8-sig") as f:
        w = csv.DictWriter(f, fieldnames=_TARGETS_FIELDS)
        w.writeheader()
        for t in targets:
            w.writerow({k: t.get(k, "") for k in _TARGETS_FIELDS})
```

- [ ] **Step 3: Implement `gui/workers/pipeline_worker.py`**

Class structure:

```python
# Signals
progress = pyqtSignal(int, int, str)   # current, total, filename
finished = pyqtSignal(dict)            # summary dict
error    = pyqtSignal(str)

# Methods
def __init__(self, scripts_dir: Path) -> None
def stop(self) -> None          # terminates subprocess
def run(self) -> None           # QThread entry: calls _run_ps1 then _run_python
def _run_ps1(self) -> int       # runs PS1, emits progress, returns total file count
def _run_python(self) -> str    # runs 02_csv_to_excel.py, returns stdout
def _parse_summary(self, stdout: str, total_files: int) -> dict
```

`_run_ps1` parses these stdout patterns from `01_extract_xic.ps1`:
- `r"Files\s+:\s+(\d+)"` — total file count
- `r"\s+\[\s*(\d+)/\s*(\d+)\]\s+(.+)"` — per-file progress

`_parse_summary` parses these stdout patterns from `02_csv_to_excel.py`:
- `r"\s+(\S+)_RT detected \(NL confirmed\): (\d+)/(\d+)"` → nl_confirmed=True
- `r"\s+(\S+)_RT detected: (\d+)/(\d+)"` → nl_confirmed=False
- `r"\s+\S+\s+OK:\d+\s+WARN:(\d+)\s+ND:\d+"` → nl_warn_count
- `r"Saved\s+:\s+(.+)"` → excel_path

Summary dict structure:
```python
{
    "total_files": int,
    "targets": [{"label": str, "detected": int, "total": int, "nl_confirmed": bool}],
    "nl_warn_count": int,
    "excel_path": str,
}
```

## Task 9: MainWindow — Assembly

**Files:** Create `gui/main_window.py`

Structure:
- `QMainWindow` with `QScrollArea` as central widget
- Container `QWidget` with `QVBoxLayout(spacing=12, margins=16)` stacking all 4 sections
- `QStatusBar` at bottom

Signal wiring:
- `settings.settings_saved` → status bar message
- `targets.targets_saved` → status bar message
- `run.run_clicked` → `_on_run()`
- `run.stop_clicked` → `_on_stop()`

`_on_run()` sequence:
1. `write_settings(self._settings.get_values())`
2. `write_targets(self._targets.get_targets())`
3. `set_enabled(False)` on settings + targets
4. `set_running(True)` on run section
5. Create `PipelineWorker(_SCRIPTS_DIR)`, connect signals, call `start()`

`_on_finished(summary)` sequence:
1. `set_complete(summary["total_files"])`
2. `set_running(False)`
3. `set_enabled(True)` on settings + targets
4. `update_results(summary)`
5. Update status bar
