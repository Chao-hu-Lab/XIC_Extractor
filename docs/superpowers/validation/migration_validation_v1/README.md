# Migration Validation v1

This directory keeps the legacy-PS1-to-Python migration validation workbook
used by `scripts/validate_migration.py`.

It is a validation artifact, not a spec. The workbook is retained because the
script default output and historical migration comparison evidence need a
stable repo path. It does not define product behavior, workbook schema,
ProductWriter authority, matrix values, selected peak/area, counted detection,
or productization tier.

